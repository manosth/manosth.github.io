Fundamentals of Computing Specialization

Machine Learning Specialization

Machine Learning is a specialization by Univeristy of Washington about Machine Learning. The premise is that students begin with an introductory course that gives a rough estimate of various Machine Learning fiels, and then subsequent courses go into depth into these fields. It consists of the following courses:

- Machine Learning Foundations: A Case Study Approach: the certificate for this course can be verified here.

Probabilistic Graphical Models Specialization

Probabilistic Graphical Models is a specialization by Stanford University about the uses and applications of graphical models. It is replica of Stanford's own graduate course on Probabilistic Graphical Models, and it is extremely demanding, one of the only specializations in Coursera tagged as "Advanced". It consists of the following courses:

- Honors Probabilistic Graphical Models 1: Representation: the certificate for this course can be verified here.

The rest of the courses are work-in-progress (I am currently taking the second one), but they are listed here for completeness.
- Probabilistic Graphical Models 2: Inference.
- Probabilistic Graphical Models 3: Learning.

Data Science at Scale Specialization

Data Science at Scale is a specialization by University of Washington about data science. It is a rigorous specialization that tries to tackle data science holistically, and provide intuition. It consists of the following courses:

- Data Manipulation at Scale: Systems and Algorithms: the certificate for this course can be verified here.
- Practical Predictive Analytics: Models and Methods: the certificate for this course can be verified here.

The rest of the courses are work-in-progress, but they are listed here for completeness.
- Communicating Data Science Results.
- Data Science at Scale - Capstone Project.
